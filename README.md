# Prueba-Técnica-

Muchas gracias por la oportunidad de participar en este proceso. Tras revisar los requerimientos de la vacante, me gustaría comentarles que he decidido desarrollar la solución técnica utilizando C# con ASP.NET Core en lugar de las tecnologías sugeridas inicialmente.

He tomado esta decisión basándome en mi especialización y dominio en el ecosistema .NET, el cual es el entorno donde cuento con mayor experiencia y donde puedo asegurar un código más robusto, eficiente y escalable para los objetivos del proyecto. Asimismo, poseo sólidos conocimientos en PHP, por lo que me siento plenamente capaz de integrar o adaptar cualquier requerimiento necesario si el proyecto lo demanda.

Además, quiero resaltar que durante el desarrollo de esta prueba he usado agentes de IA como herramienta de apoyo estratégico. Considero fundamental el dominio de estas tecnologías en el desarrollo de software actual; las utilizo no solo para optimizar mis tiempos de entrega y resolver problemas de lógica, sino para validar mejores prácticas, asegurar la limpieza del código y mantener una arquitectura consistente. Mi enfoque con la IA es el de un desarrollador que sabe dirigir y supervisar la herramienta para maximizar la calidad del producto final.
